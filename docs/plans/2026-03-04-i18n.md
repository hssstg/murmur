# i18n Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Make all hardcoded Chinese UI strings localizable so the app displays in English on English-locale macOS systems.

**Architecture:** SPM localized resources (.lproj/Localizable.strings). Chinese strings used as default; English provided via en.lproj. SwiftUI Text() / LocalizedStringKey literals in views work automatically once the .strings files are in place. NSMenuItem and programmatic strings use `String(localized:)`. Format strings use `String(format: String(localized: "key"), arg)`.

**Tech Stack:** Swift 6, SwiftUI, Swift Package Manager, NSLocalizedString / String(localized:) API.

---

## Quick Reference

### How localization works in this project

**SwiftUI (automatic):** All these accept `LocalizedStringKey` as their first argument and auto-look-up at runtime:
- `Text("key")`, `Section("key")`, `Toggle("key", isOn:)`, `Picker("key", selection:)`, `TextField("key", text:)`, `Button("key")`, `.help("key")`, `.confirmationDialog("key", ...)`, `Label("key", systemImage:)`

Just change the string literal from Chinese to the key, e.g. `Text("快捷键")` → `Text("settings.hotkey.section")`.

**AppKit / programmatic (manual):**
```swift
String(localized: "menu.history")           // simple key
String(format: String(localized: "hotwords.count"), n)  // %d format arg
```

**Format strings in .strings files:** Use `%d` for Int, `%@` for String.

**State var holding a display label:**
```swift
@State private var saveLabel: LocalizedStringKey = "settings.save"
// later: saveLabel = "settings.saved"
// in view: Button(saveLabel) { ... }
```

**Step 1: Refactor TimeFilter enum**

The current rawValue is Chinese and is used directly in `Text($0.rawValue)`. Change rawValue to an English identifier and add a `displayName` property:

```swift
enum TimeFilter: String, CaseIterable {
    case today  = "today"
    case week   = "week"
    case month  = "month"
    case all    = "all"

    var displayName: LocalizedStringKey {
        switch self {
        case .today: "history.filter.today"
        case .week:  "history.filter.week"
        case .month: "history.filter.month"
        case .all:   "history.filter.all"
        }
    }

    func startDate() -> Date? {
        let cal = Calendar.current
        switch self {
        case .today: return cal.startOfDay(for: Date())
        case .week:  return cal.date(byAdding: .day, value: -7,  to: Date())
        case .month: return cal.date(byAdding: .day, value: -30, to: Date())
        case .all:   return nil
        }
    }
}
```
