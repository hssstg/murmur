特性

用声音工作，不打断思路

🎙
Push-to-Talk

按住热键录音，松键即触发识别，结果直接插入当前光标所在位置。

〰
Siri 风格波形

录音期间浮窗显示 60fps 平滑波形动画，不抢焦点，不遮挡内容。

✦
LLM 归整

可选接入 OpenAI 兼容 API，自动去语气词、修正同音错字、整理标点。

📋
历史记录

所有识别结果持久化存储，支持手动修正，原文与修正版本均留档备查。

📚
智能热词库

同步火山引擎自学习平台，AI 从历史修正中自动提取专有名词建议。

⌨
全热键支持

Option、Control、CapsLock、F13–F15 及鼠标侧键均可配置为录音触发键。

使用流程

三步开始语音输入

01
配置凭证

点击菜单栏图标 → 设置，填写火山引擎 BigASR 的 App ID 与 Access Token。LLM 归整和热词管理均为可选项。

02
授予权限

系统设置 → 隐私与安全 → 辅助功能，将 Murmur 加入允许列表。用于热键监听与文字插入。

03
按住说话

将光标放在任意文本框，按住 ⌥ Right Option，说话，松开。识别文字直接出现在光标位置。
